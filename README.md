# Advent of Code

- Supports multiple years and days.
- Supports multiple languages.
  - ruby
  - python
  - typescript (with Bun)
  - ...

## Setup

```bash
# Setup the year 2024
meta/setup.rb 2024
```

## Usage

```bash
# Run and submit the solution for year 2024, day 4, level 2
meta/run.rb 2024 4 2
```
